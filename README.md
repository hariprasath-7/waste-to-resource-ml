# Waste-to-Resource ML

AI-powered waste classification and reuse recommendation system using Machine Learning, Computer Vision, and EfficientNet CNN architecture.

## Project Overview

Waste-to-Resource is an AI/ML-based web application designed to classify waste materials and provide smart reuse or recycling recommendations to support sustainable waste management and circular economy practices.

The system uses deep learning and image processing techniques to identify different categories of waste materials from uploaded images and helps users discover practical reuse ideas.

## Features

* AI-powered waste classification
* Image-based waste detection
* EfficientNet CNN model integration
* Real-time prediction system
* Reuse and recycling recommendations
* Flask web application
* User-friendly interface
* Sustainable waste management support

## Waste Categories

The model currently classifies:

* Plastic Bottles
* Glass Bottles
* Cardboard
* Aluminum Cans
* Cloth Waste
